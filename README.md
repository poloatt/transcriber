# Transcriber App

## Docker Environments

### Development
```bash
docker-compose up
```

### Testing
```bash
docker-compose -f docker-compose.test.yml up
```

### Production
```bash
docker-compose -f docker-compose.prod.yml up -d
```

To install the necessary dependencies, run:
```bash
npm install
```

## Usage

To start the application, use:
```bash

```
